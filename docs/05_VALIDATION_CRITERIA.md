# Neon Pop Sweeper - Prototype Validation Criteria v2.3

이 문서는 프로토타입 완성 여부를 판단하기 위한 기능 체크리스트와 판정 규칙을 정의합니다.

## 1) 기능 완료 체크리스트
- [ ] 시작 패널 -> `게임 시작` -> 조인 씬 -> `게임 시작!` 오버레이 -> 정상 라운드 진입
- [ ] `게임 시작` 버튼은 시작 카드 내부 중앙, `전체화면` 버튼은 시작 카드 외부 하단 중앙에 분리 배치됨
- [ ] 조인 씬: 약 5초 진행 바·남은 시간·합류 피드 표시, 미합류 AI는 화면에 보이지 않음, 합류 시 화면 밖에서 슬라이드 인
- [ ] 조인 씬/게임 시작 전에는 AI 포함 액터의 실제 이동이 발생하지 않음(합류 슬라이드 인만 표시)
- [ ] 조이스틱 + WASD/방향키 이동 모두 동작
- [ ] 네온 사각형 닷 수집/리스폰 정상 동작
- [ ] 닷 컬러별 획득 점수가 스펙과 일치함(예: 기본 청록 `+10`, 보너스 연두 `+25`, 피버 핑크 `+40`, 체이서 보라 `+55`, 위험 빨강 `0` + 5% 손실 규칙)
- [ ] 움직이는 보라색 닷(`chaser`)이 스폰/이동/소멸/수집 보상으로 정상 동작
- [ ] 붉은 삼각 위험 닷(`hazard`) 획득 시 획득 대상이 빨갛게 점멸하고 `floor(score×5%)`(보유 점수 한도) 손실이 적용됨
- [ ] 위험 닷으로 잃은 점수만큼 필드에 환원 닷이 분산 스폰되고, 발사 비행 구간에는 수집되지 않음
- [ ] 붉은 위험 닷(`hazard`)의 동시 활성 개수가 최대 `3`을 넘지 않음
- [ ] AI 이동 로직이 위험 닷(`hazard`) 인지/회피를 별도 수행하지 않음
- [ ] AI 닉네임이 유저형 이름 규칙을 따르고 기계형 이름(`Bot_1`, `AI_Player`)을 사용하지 않음
- [ ] 액터 닉네임 라벨(YOU + AI)이 월드에서 캐릭터 머리 위에 표시됨(비합류 봇 제외)
- [ ] 닷 획득 시 플레이어 크기가 증가(성장)하는 동작 확인
- [ ] 닷을 일정 시간 획득하지 않으면 플레이어 크기가 점진적으로 감소하는 동작 확인
- [ ] 플레이어 크기 성장값은 최대치(`scaleLevel = 50`)를 넘지 않음
- [ ] 콤보 0.6초 규칙 확인(0.6초 내 미획득 시 콤보 즉시 `0`, 체인 초기화)
- [ ] 콤보 색상/크기 단계 연출(15단위, 10% 증분) 확인
- [ ] 콤보 증가 시 텍스트 스케일 연출이 1회만 재생되고, 해당 단계 기본 크기 대비 최대 2배까지 확대 후 복귀함
- [ ] 닷 수집 시 닷 타입에 따른 파티클 연출이 구분되어 보임
- [ ] 위험 닷 환원 분산의 퍼짐 거리가 지나치게 짧지 않음
- [ ] 1등 왕관이 닉네임 라벨과 겹치지 않음
- [ ] 60초 정확 종료 및 결과 패널 표시
- [ ] 랭킹/HUD/결과 패널 점수가 정수로 표시됨(소수점 없음)
- [ ] 랭킹 UI가 `playing` 단계에서만 표시되고 시작/조인/시작연출/결과 단계에는 표시되지 않음
- [ ] 랭킹 패널이 게임 뷰(16:9) 좌측 상단 기준에 고정되고, 화면비 변경 시에도 게임 뷰 경계를 벗어나지 않음
- [ ] 랭킹 목록에서 플레이어(`YOU`) 항목이 노란색으로 강조 표시됨
- [ ] Restart 후 초기화/재플레이 정상 동작
- [ ] 치명적 런타임 오류 없이(콘솔 에러/예외로 게임 중단 없음) 1회 이상 완주 가능
- [ ] 초기 로드/리사이즈 단계에서 `undefined.style` 계열 콘솔 에러가 발생하지 않음
- [ ] 게임 뷰 축소 시 HUD/오버레이/UI가 함께 비례 축소되고 최소 스케일 고정 문제로 과대 표시되지 않음

## 2) 판정 규칙

- 기능 체크리스트 전부 통과 시 "프로토타입 합격"으로 판정
- 미달 시, 수치 변경 전에 원인을 입력/카메라/UI 가독성/스폰 밀도 축으로 먼저 분류

## 3) KPI(정량 검증 지표)

- KPI-1. 완주 안정성: 연속 3회 플레이 시 3/3회 모두 60초 라운드를 오류 없이 완주
- KPI-2. 입력 신뢰성: 각 세션 시작 후 10초 이내 플레이어 이동 입력(포인터 또는 키보드)이 최소 1회 이상 정상 반영
- KPI-3. 재시작 안정성: Restart 3회 반복 후에도 시작 패널 -> 조인 -> 라운드 진입 플로우가 매회 정상 동작
- KPI-4. 점수 표시 일관성: 랭킹/HUD/결과 패널의 점수가 전 구간 정수(`floor`) 표기 유지
- KPI-5. 경쟁 가시성: 매 세션에서 플레이어+AI 총 4개 닉네임 라벨이 월드에 식별 가능하게 표시됨(비합류 구간 제외)
